# NovaZ Experiments

This directory is for messy, focused experiments while the Nova Z-machine
runtime is taking shape.

Good candidates:

- XRAM window access probes
- Z-machine header/checksum readers
- Z-text decoder sketches
- opcode decoder experiments
- dictionary/tokenizer tests
- VGC status-line and scroll-region tests
- notes from running Zork I/II/III under the Nova runtime shell

Keep commercial story files out of the repository. Use local ignored project
paths such as `../projects/zork-i/STORY.BIN` when an experiment needs a story
file.
