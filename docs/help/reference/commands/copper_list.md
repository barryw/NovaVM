---
title: "COPPER_LIST"
type: command
category: Graphics
keywords: [copper_list, graphics]
syntax: "COPPER LIST n"
see_also: [graphics-and-display]
---

Set target list to *n* (0--127). Subsequent ADD/CLEAR edit this list. Reset target list back to the active list.

See the [Graphics And Display](/docs/help/guides/graphics-and-display.md) guide for more details.
