---
title: "XRESET"
type: command
category: Memory
keywords: [xreset, memory]
syntax: "XRESET"
see_also: [expansion-memory]
---

Clear all XRAM allocation and named-block state. `XRESET` does not need to erase
every raw byte in XRAM, but all names, handles, and page-usage records are
discarded.

See the [Expansion Memory](/docs/help/guides/expansion-memory.md) guide for more details.
