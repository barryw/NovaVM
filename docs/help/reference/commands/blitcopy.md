category: DMA

syntax: "BLITCOPY ss,sa,sstr,ds,da,dstr,w,h"

Copy rectangle: *w* bytes wide, *h* rows, with source stride *sstr* and destination stride *dstr*.

See the [Dma And Blitter](/docs/help/guides/dma-and-blitter.md) guide for more details.
