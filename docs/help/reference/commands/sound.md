---
title: "SOUND"
type: command
category: Sound
keywords: [sound]
syntax: "SOUND note,dur[,inst]"
see_also: [sound-and-music]
---

Play MIDI *note* (0--127) for *dur* frames (1/60 s). Optional *inst* selects instrument preset (0--15, default 0).

See the [Sound And Music](/docs/help/guides/sound-and-music.md) guide for more details.
