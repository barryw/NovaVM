---
title: "XPEEK"
type: command
category: Memory
keywords: [xpeek, memory]
syntax: "XPEEK(offset)"
see_also: [expansion-memory]
---

Read one byte from *offset* in the active `XBANK`. `XPEEK` is a numeric
function and can be used inside expressions.

See the [Expansion Memory](/docs/help/guides/expansion-memory.md) guide for more details.
