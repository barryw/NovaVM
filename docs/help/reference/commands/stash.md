---
title: "STASH"
type: command
category: Memory
keywords: [stash, memory]
syntax: "STASH ram,offset,length"
see_also: [expansion-memory]
---

Copy *length* bytes from CPU RAM to XRAM (raw, no name). Store a named XRAM block from CPU RAM.

See the [Expansion Memory](/docs/help/guides/expansion-memory.md) guide for more details.
