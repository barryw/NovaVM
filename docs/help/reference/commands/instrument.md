---
title: "INSTRUMENT"
type: command
category: Sound
keywords: [instrument, sound]
syntax: "INSTRUMENT id,wave,a,d,s,r"
see_also: [sound-and-music]
---

Define instrument preset *id* (0--15). *wave*: 10=tri,20=saw, 40=pulse,80=noise. *a,d,s,r*: ADSR values 0--15.

See the [Sound And Music](/docs/help/guides/sound-and-music.md) guide for more details.
