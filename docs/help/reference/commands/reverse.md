---
title: "REVERSE"
type: command
category: Display
keywords: [reverse, inverse, text, display]
syntax: "REVERSE [fg,bg]"
see_also: [reverseoff, flash, flashoff, color]
---

Turn on reverse text for subsequent printed characters.

Without arguments, NovaBASIC swaps the current text foreground and background.
With `fg,bg`, reverse text uses those explicit colors instead.
