---
title: "SPRITEDATA"
type: command
category: Sprites
keywords: [spritedata, sprites]
syntax: "SPRITEDATA n,row,b1...b8"
see_also: [graphics-and-display]
---

Define one row of sprite shape data (8 bytes, 16 pixels). Bytes b3--b8 are optional; omitted bytes leave those pixels unchanged.

See the [Graphics And Display](/docs/help/guides/graphics-and-display.md) guide for more details.
