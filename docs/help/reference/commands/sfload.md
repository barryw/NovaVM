---
title: "SFLOAD"
type: command
category: Sound
keywords: [sfload, soundfont, midi]
syntax: "SFLOAD \"filename\""
see_also: [midplay, sound-and-music]
---

Load a soundfont for MIDI playback.

```basic
10 SFLOAD "piano.sf2"
20 MIDPLAY "song.mid"
```

See the [Sound And Music](/docs/help/guides/sound-and-music.md) guide for more details.
