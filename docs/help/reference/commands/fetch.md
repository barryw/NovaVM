---
title: "FETCH"
type: command
category: Memory
keywords: [fetch, memory]
syntax: "FETCH ram,offset,length"
see_also: [expansion-memory]
---

Copy *length* bytes from XRAM to CPU RAM (raw). Load a named XRAM block back into CPU RAM.

See the [Expansion Memory](/docs/help/guides/expansion-memory.md) guide for more details.
