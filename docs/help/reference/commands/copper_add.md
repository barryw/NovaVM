---
title: "COPPER_ADD"
type: command
category: Graphics
keywords: [copper_add, graphics]
syntax: "COPPER ADD x,y,reg,val"
see_also: [graphics-and-display]
---

Add copper event: set *reg* to *val* at position (*x*,*y*). Registers: BGCOL, MODE, SCROLLX, SCROLLY, SPRX(*n*), SPRXH(*n*), SPRY(*n*), SPRYH(*n*), SPRSHAPE(*n*), SPRFLAGS(*n*), SPRPRI(*n*).

See the [Graphics And Display](/docs/help/guides/graphics-and-display.md) guide for more details.
